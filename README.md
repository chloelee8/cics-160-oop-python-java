# Object-Oriented Programming in Python (CICS 160)

This repository contains coursework from **CICS 160 – Object-Oriented Programming** at UMass Amherst.

## Topics & Skills
- Object-oriented programming (classes, objects, inheritance)
- Functions and modular design
- Basic data structures (lists, dictionaries)
- Problem-solving and algorithmic thinking
- Debugging and testing

## Notes
This course emphasizes object-oriented programming in Python, including designing classes, using inheritance, and developing modular, well-structured programs.
